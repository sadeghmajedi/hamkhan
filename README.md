# هم‌خوان — دستیار روخوانی و دیکته (نسخه MVP v0.1.0)

## نصب و اجرا
همه‌چیز در یک پوشه‌ی ساده است، دقیقاً مثل Shakhes Planner و PFM:
1. این پوشه را در یک ریپوی GitHub جدید (یا موجود) قرار بده، مثلاً `sadeghmajedi.github.io/hamkhan/`
2. GitHub Pages را برای همان ریپو فعال کن
3. آدرس را روی iPad در **سافاری** باز کن (نه کروم — چون تشخیص گفتار فقط در سافاری/وب‌کیت کار می‌کند)

## نکته‌ی مهم: Add to Home Screen
اگر برنامه را با «Add to Home Screen» نصب کنی، بخش **روخوانی** (تشخیص گفتار) اصلاً کار نمی‌کند — این یک محدودیت خود Safari روی iOS/iPadOS است، نه باگ برنامه. توصیه:
- برای استفاده‌ی روزمره، یک **بوکمارک** در سافاری بساز (نه آیکون Home Screen)
- بخش تنظیمات این موضوع را به‌طور خودکار تشخیص می‌دهد و به کاربر هشدار می‌دهد

## ساختار فایل‌ها
- `index.html` — کل برنامه (HTML+CSS+JS در یک فایل، طبق الگوی PFM)
- `manifest.json` — برای ظاهر Add to Home Screen
- `sw.js` — کش کردن پوسته‌ی برنامه برای اجرای آفلاین صفحات غیر از روخوانی
- `version.json` — برای قابلیت «بررسی نسخه» در تنظیمات؛ هر بار که نسخه جدید منتشر می‌کنی، این فایل را هم به‌روز کن (مطابق الگوی version.json در دو پروژه‌ی دیگر)
- `icon-192.png` / `icon-512.png` — آیکون‌های ساده‌ی موقت؛ می‌تونی بعداً با آیکون طراحی‌شده جایگزین کنی

## ذخیره‌سازی داده
همه‌چیز (تاریخچه‌ی جلسات، متون دلخواه) در `localStorage` با کلید `hamkhan_db_v1` ذخیره می‌شود — کاملاً محلی روی همان دستگاه/مرورگر. برای انتقال بین دستگاه‌ها یا جلوگیری از پاک شدن هنگام خالی کردن کش سافاری، از دکمه‌ی «خروجی گرفتن» در تنظیمات به‌طور منظم استفاده کن.

## محدودیت شناخته‌شده: تشخیص گفتار فارسی
`webkitSpeechRecognition` در سافاری کار می‌کند ولی:
- نیاز به اینترنت دارد (صدا برای پردازش به سرور اپل فرستاده می‌شود)
- دقتش برای صدای کودک ممکن است نسبت به بزرگسال کمتر باشد — بعد از چند بار تست با فرزندتان، اگر دقت پایین بود، می‌شود موتور را در آینده به Azure Speech یا Whisper (که دقت بهتری برای فارسی دارند ولی نیاز به سرویس ابری/کلید API دارند) تغییر داد.

## گام بعدی پیشنهادی
بعد از تست چند روزه با فرزندتان روی iPad واقعی، بهتره بررسی کنیم:
1. دقت تشخیص گفتار فارسی روی صدای کودک ۸ ساله چطوره
2. آیا نیاز به تنظیم حساسیت الگوریتم مقایسه‌ی کلمات (`alignWords`) هست یا نه (مثلاً برای کلماتی که فقط یک حرف تفاوت دارند)
3. افزودن صداهای TTS فارسی بهتر در صورت نیاز (چون صدای سیستمی سافاری برای فارسی محدود است)
